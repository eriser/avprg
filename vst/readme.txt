Installation der Beispielplugins:

1. VST-SDK von Steinberg herunterladen

http://www.steinberg.net/sdk_downloads/vstsdk366_27_06_2016_build_61.zip

2. entpacken

3. darin neuen Unterordner anlegen

VST3 SDK\public.sdk\avprg

4. dorthin diese Dateien kopieren bzw. dort git clone aufrufen

5. zur Überprüfung: das Projekt "PluginTemplate" sollte folgenden Pfad haben:

VST3 SDK\public.sdk\avprg\vst\PluginTemplate
